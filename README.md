программа на go для загрузки и редактирования файлов на сервере, есть web интерфейс который можно настраивать, показывает миниатюры, по клику открывает в отдельном окне фото и видео. запускается ./filemanager далее заходим в баузер по адресу 127.0.0.1:8080 и радуемся. Программа минималистична и необходима для управлления хранилищем фото и видео, выполняет все необходимые команды, копировать, вставить, переместить, загрузить и скачать + просмотр.
